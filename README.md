# 💰 מערכת ניהול משק בית

מערכת FullStack מקיפה לניהול תקציב משק בית עם אפשרויות מתקדמות למעקב אחר הכנסות, הוצאות, חסכונות והתחייבויות.

## 🚀 טכנולוגיות

### Frontend
- **React 18** + **Vite** - ממשק משתמש מהיר ומודרני
- **Material-UI (MUI)** - עיצוב מקצועי ורספונסיבי
- **Redux Toolkit** - ניהול state מתקדם
- **Recharts** - גרפים ויזואליים
- **React Hook Form + Yup** - טפסים עם validation
- **Axios** - תקשורת עם השרת
- **Date-fns** - ניהול תאריכים

### Backend
- **Node.js** + **Express** - שרת מהיר ויעיל
- **MongoDB** + **Mongoose** - מסד נתונים NoSQL
- **JWT** - אימות מאובטח
- **Bcrypt** - הצפנת סיסמאות

### 📊 לוח בקרה
- סיכום כספי מקיף
- 4 כרטיסי סטטיסטיקה (הכנסות, הוצאות, יתרה, חסכונות)

## 🛠️ התקנה והרצה

### דרישות מקדימות
- Node.js (גרסה 18 ומעלה)
- MongoDB (מותקן מקומית או Atlas)
- npm או yarn

### התקנת Backend

```bash
cd server
npm install
```

צור קובץ `.env` בתיקיית `server`:
```env
PORT=5000
MONGO_URI=mongodb://localhost:27017/household-budget
JWT_SECRET=your_super_secret_jwt_key_change_this_in_production
JWT_EXPIRE=30d
NODE_ENV=development
```

הרץ את השרת:
```bash
npm run dev
```

### התקנת Frontend

```bash
cd client
npm install
```

הרץ את הממשק:
```bash
npm run dev
```

הממשק יהיה זמין בכתובת: `http://localhost:5173`

## 🎯 דרכי שימוש מומלצות

1. **התחל עם הרשמה** - צור חשבון וקבל אוטומטית קטגוריות ברירת מחדל
2. **הגדר יעדים** - עבור ל"יעדי תקציב" והגדר תקציב חודשי לכל קטגוריה
3. **הוסף תנועות** - השתמש ב-FAB (כפתור צף) להוספה מהירה
4. **עקוב אחר היתרה** - בעת הוספת הוצאה, ראה את תיבת היתרה החיה
5. **צור חסכונות** - הגדר יעדי חיסכון ועקוב אחר ההתקדמות
6. **נהל התחייבויות** - רשום הלוואות ועקוב אחר התשלומים
7. **בדוק המלצות** - קבל המלצות חכמות בלוח הבקרה
